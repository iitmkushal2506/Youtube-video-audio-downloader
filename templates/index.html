About This Project

This YouTube Video Downloader is a web-based application built using Python Flask that allows users to download YouTube videos and audio by simply pasting the video link. The application processes the URL, extracts video information, and provides download options in MP4 and MP3 formats.

This project is designed to run locally on your machine. Due to platform restrictions and YouTube policies, this application is not intended for deployment on public hosting platforms. It works best when executed locally.

Purpose of This Project

This project was developed as part of learning backend development using Python Flask. It demonstrates how frontend and backend systems interact to process user inputs, retrieve external content, and serve downloadable files dynamically.

Technologies Used

  • Python
  • Flask Framework
  • yt-dlp
  • HTML
  • CSS

Concepts Implemented

  • Backend Routing
  • Form Handling

  • Video Metadata Extraction
  • File Handling and Streaming
  • Responsive UI Design

How It Works

Users paste a YouTube link into the input box and click fetch video. The backend uses yt-dlp to extract video information, displays a preview, and provides download options. Once the user clicks download, the server processes the request and sends the file to the user.

Run This Project Locally

Follow these steps to run this project locally after cloning from GitHub:

  • Clone the repository from GitHub
  • Open the project folder in your code editor
  • Install dependencies using: pip install flask yt-dlp
  • Make sure Python is installed on your system
  • Run the app using: python app.py
  • Open browser and go to: http://127.0.0.1:5000

Note

If the video preview is not playable after fetching, do not worry. This may happen due to YouTube embed restrictions on certain videos. You can still download the video or audio using the download buttons provided.
